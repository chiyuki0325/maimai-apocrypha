# BLACK ROSEちほー11（blackrose11 区域）
::: info 区域简介
既是炼金术师又是侦探的 Owl。
在与追查黑蔷薇病的过程中结识的伙伴们一起，不断解开种种谜团。
本次是关于将世界卷入战火的『福音』的故事。
:::
::: tip 一些小信息
Tristan 蒙受的杀人嫌疑。追查此事的 Owl 侦探事务所一行人的去向是？
:::

黒薔薇病を追う内に出会った仲間たちと、様々な謎を解明していく。
今回は、世界を戦禍に巻き込む『福音』についての物語。
